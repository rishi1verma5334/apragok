# APRA: Group of Knowledge — Official Website

> The official web presence for **APRA: Group of Knowledge (APRAGOK)** — built for clarity, speed, and professional impact.

---

## 🌐 Overview

This is the official website for **APRA: Group of Knowledge**, designed to represent the company's identity, services, and vision online. The site is built with a modern, performant tech stack to ensure a fast, responsive, and visually polished experience across all devices.

---

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| **React** | Component-based UI development |
| **TypeScript** | Type-safe, maintainable codebase |
| **Vite** | Lightning-fast build tool and dev server |
| **Tailwind CSS** | Utility-first styling for rapid UI design |
| **shadcn/ui** | Accessible, customizable UI components |

---

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or above recommended)
- npm or yarn

## 📁 Project Structure

```
apragok/
├── public/          # Static assets
├── src/
│   ├── components/  # Reusable UI components
│   ├── pages/       # Page-level components
│   ├── assets/      # Images, icons, fonts
│   └── main.tsx     # App entry point
├── index.html
├── tailwind.config.ts
├── vite.config.ts
└── package.json
```
